# 流体汉字 | Flux

> 本项目现仅为 Alpha 开发阶段

**流体汉字 (Flux)** 是一款基于 Web 技术的汉字解构与动态可视化交互应用。它不仅能够动态展示汉字的标准书写轨迹，还提供了独特的“笔顺虚影对比”功能，将汉字拆解为逐笔进阶的静态步骤图，辅助书法爱好者和学习者深入理解汉字的间架结构。

## ✨ 核心特性

- 动态书写仿真：利用 SVG 路径算法模拟真实的汉字书写顺序与节奏。

- 笔顺虚影对比：在步骤拆解图中，当前笔画高亮显示，其余笔画以虚影形式衬托，辅助定位。

- 智能字典引擎：

  - 内置精选文化汉字数据库，包含深度释义与字源解读。

  - 支持多音字拼音自动标注。

  - 针对库外汉字提供基于字符编码的“部首推导”逻辑。

- 沉浸式视觉设计：采用毛玻璃交互界面，配以流体粒子背景系统。

- 无极缩放：全矢量路径渲染，在任何分辨率下均保持锐利清晰。

## 🚀 技术实现方案

本项目基于以下核心技术构建：

1. 路径解析：通过 Hanzi Writer 驱动矢量路径数据。

2. 动画调度：使用 GSAP (GreenSock) 负责 UI 状态切换与粒子动力学。

3. 语言学处理：集成 pinyin-pro 实现精准的拼音解析。

4. 渲染算法：

  - Stroke Dash Offset: 实现“白描”式的书写动画效果。

  - Index-based Filtering: 实现静态步骤图中的虚实对比渲染。

## 🛠️ 快速开始

### 本地运行（但由于需要连接数据库等，运行时需联网）

本项目为单文件架构（Single-File Architecture），无需复杂的构建环境，直接打开 index.html 即可运行。

克隆仓库：

git clone [https://github.com/your-username/flux-hanzi.git](https://github.com/your-username/flux-hanzi.git)


进入目录：

cd flux-hanzi


在浏览器中打开 index.html。

## 📖 离线字库示例

项目内置了如“永”、“道”、“德”、“爱”等具有书法代表性的汉字。对于这些汉字，应用将展示详尽的【象形/会意】解读及书法背景。
